# 🎁 礼物参谋 (Gift Advisor)

## 使用场景
- 用户说"送什么礼物好"、"XX节快到了不知道送什么"
- 用户提到节日/纪念日/生日需要礼物建议
- 用户想通过礼物推进关系或修复关系

## 工作流程

### Step 1: 信息收集
主动询问以下关键信息（未知的才问，已知的直接使用）：

| 信息项 | 原因 |
|---|---|
| **关系阶段** | 暗恋/刚表白/热恋期/稳定期/挽回 |
| **送礼场景** | 生日/七夕/情人节/纪念日/随手礼 |
| **对方特征** | 性格/爱好/职业/生活习惯 |
| **预算范围** | 不超过xxx元 |
| **你的目的** | 表达心意/推进关系/制造惊喜/修复关系 |

### Step 3: 深化建议（可选）
如果用户选定了某个方向，进一步提供：
- **包装建议**: 如何提升视觉冲击
- **时机建议**: 什么时候送效果最好
- **跟进策略**: 送完礼物之后怎么接话

## 核心原则

### 礼物的三个价值维度
1. **实用价值**: 对方真正用得上
2. **情感价值**: 证明"你了解她"
3. **社交价值**: 值得她发朋友圈

### 关系阶段对应策略
| 阶段 | 礼物策略 |
|---|---|
| **暗恋/追求期** | 有心意但不太贵，避免压力感 |
| **热恋期** | 可以贵，但要有惊喜感和仪式感 |
| **稳定期** | 实用 + 情感双轨并行 |
| **挽回/修复** | 低调真诚，避免用钱砸 |

### 禁忌礼物类型
- 现金/购物卡（太敷衍）
- 包含"永久/一生"字眼的物品（太早）
- 对方明确说不想要的东西
- 自己用惯了但对方不一定喜欢的品牌
