# 🔍 全网搜索 (Web Search)

## 使用场景
- 用户问"最新的恋爱心理学研究说什么"
- 用户问某地的文化习俗（相关法律/规范）
- 需要实时餐厅/活动资讯
- 用户提到不确定的事实，需要核实

## 工作流程

### Step 1: 判断是否需要搜索
以下情况**必须**调用搜索：
- 需要近1年内的信息
- 涉及特定地点/政策/人物的具体事实
- 用户明确说"搜一下"、"查一查"
- 自身知识截止日期之后的事件

### Step 2: 构造精确搜索词
- 用**中文关键词**，去掉无意义词汇
- 加上时间限定（如"2024"）提升时效性
- 复杂问题拆分为多次搜索

调用 `web_search` 工具，传入 `query` 参数。

### Step 3: 消化与整合
- 不要直接粘贴搜索结果给用户
- 将结果**提炼、整合**成完整的答案
- 如果多个来源有冲突，说明并给出倾向性判断
- 提供信息来源的链接

## 输出格式
直接用自然语言回答，在结尾附带来源：

```
[正式回答内容]

📖 参考来源:
- [来源1标题](URL)
- [来源2标题](URL)
```
