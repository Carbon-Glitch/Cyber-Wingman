# 🔮 平行宇宙 (Parallel Universe)

## 使用场景
- 用户说"如果当时我这样说就好了"
- 用户分享了一段让结果很糟糕的对话，想知道最优解
- 用户说"帮我重写这段对话"、"假设我当时回复了XXX"

## 工作流程

### Step 1: 解析"当前宇宙"
让用户描述（或粘贴）发生了什么：
- 当时的场景/背景
- 对方说了什么
- 用户实际回复了什么
- 结果怎样（对方冷了？生气了？失去兴趣？）

### Step 2: 诊断失误点
分析"当前宇宙"中的问题：
- **失分动作**: 哪句话说错了，为什么错？
- **心理模型误判**: 用户对对方的理解偏差在哪里？
- **机会窗口**: 当时最佳的切入点在什么时机？

### Step 3: 生成平行宇宙
模拟 **3 条分叉时间线**，推演不同回复策略的结局：

```
🌌 平行宇宙推演 — [场景简介]

━━━ 当前宇宙（已发生）━━━
对方: [原话]
你:   [你的实际回复]
结果: [实际结果]
失误: [核心问题诊断]

━━━ 宇宙 A · 高情商路线 ━━━
你:   "[最优回复]"
对方反应预测: [TA 大概率会...]
后续走向: [关系升温/话题延伸/留下悬念...]
情商解析: [为什么这样回更好]

━━━ 宇宙 B · 反差拉力路线 ━━━
你:   "[制造好奇/若即若离的回复]"
对方反应预测: [TA 可能会...]
后续走向: [...]
策略解析: [...]

━━━ 宇宙 C · 坦诚破冰路线 ━━━
你:   "[直接、真实、有温度的回复]"
对方反应预测: [...]
后续走向: [...]
策略解析: [...]

━━━ 现在怎么办？━━━
当前宇宙已经发生，但可以用以下话术"重置进度"：
💬 补救话术: "[一句话挽回建议]"
```

## 核心原则
- **不评判用户**，只分析策略对错
- **平行宇宙必须可执行**，不是空想，是真实可复制的话术
- **结合对方性格特点**给出个性化预测，而不是通用模板
- 每个宇宙的回复要**有明显差异**，代表不同策略思路
- 最后的"补救话术"需要考虑时间差，如果时间过了一天以上，需要自然的话题切入口
