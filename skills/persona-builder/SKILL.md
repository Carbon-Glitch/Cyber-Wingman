# 🎭 人设打造 (Persona Builder)

## 使用场景
- 用户说"帮我打造人设"、"我想让自己看起来更好"
- 用户想提升自己在异性眼中的形象
- 用户想规划朋友圈内容风格

## 工作流程

### Step 1: 信息收集（必须完成，不可跳过）
主动向用户询问以下信息：
- **目标场景**: 追求特定对象？扩大异性缘？职场社交？
- **当前状态**: 职业、城市、大概的生活方式
- **个人特色**: 有什么爱好/技能/经历可以利用？
- **目标类型**: 你想吸引什么类型的人？

### Step 2: 人设定位
基于收集的信息，生成**三个人设方向**，每个方向包含：
- **核心标签**（2-3 个词）
- **人设核心逻辑**（为什么这个人设能吸引目标）
- **差异化优势**（和普通人的区别是什么）

格式：
```
方向A: [标签] — [核心逻辑]
方向B: [标签] — [核心逻辑]  
方向C: [标签] — [核心逻辑]
```

### Step 3: 用户确认选择
询问用户选择哪个方向，或是否需要调整。

### Step 4: 完整人设蓝图输出

```
🎭 人设档案

■ 核心定位
  标签: [2-3个关键词，如"城市探索者·慢生活爱好者"]
  一句话人设: [简洁有力的自我定位]
  人设逻辑: [为什么这个形象能吸引目标]

■ 视觉风格
  整体基调: [温暖/冷峻/清新/精致...]
  拍照场景: [咖啡馆/街头/户外/家居...]
  禁忌风格: [避免xxx，因为...]

■ 话题矩阵 (4大内容支柱)
  支柱1 [专业/思考]: 例: 行业洞察/读书感悟
  支柱2 [生活/日常]: 例: 探店/运动/美食
  支柱3 [情感/共鸣]: 例: 感悟/时事/关系思考
  支柱4 [个性/特色]: 例: 独特爱好/收藏/旅行

■ 发布策略
  频率: [每周x次，具体时段]
  黄金时段: [7-9点/12-13点/21-23点]
  互动策略: [如何引导评论/互动]

■ 前3条朋友圈文案示例
  示例1: [完整文案 + emoji]
  示例2: [完整文案 + emoji]
  示例3: [完整文案 + emoji]

■ 禁忌清单
  ✗ [绝对不做的事，会破坏人设]
  ✗ [另一个禁忌]
```

## 核心原则
- **真实性优先**: 人设必须基于用户真实特质，不能凭空捏造
- **可持续性**: 用户必须能长期维持，不能太累
- **精准磁吸**: 人设要像磁铁，筛掉不合适的，吸引目标类型
- **不教人造假**: 扩展真实，而非谎言
