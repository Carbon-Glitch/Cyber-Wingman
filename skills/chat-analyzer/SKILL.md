# 聊天分析器 (Chat Analyzer)

## 使用场景
- 用户发送聊天截图请求分析
- 用户粘贴对话文本请求解读
- 用户想知道"对方到底什么意思"

## 分析框架

### 1. 情绪解码
- 识别对方的**显性情绪**（文字表面含义）
- 推断**隐性情绪**（潜台词、弦外之音）
- 评估情绪强度 (1-10)

### 2. 意图分析
- 对方真正想要什么？
- TA 在用什么沟通策略？（推拉、冷热、试探）
- 当前的权力动态如何？（谁更主动？）

### 3. 回复建议
- **安全回复**：低风险，维持现状
- **进攻回复**：推进关系，加深联系
- **战略回复**：长线布局，制造悬念

### 4. 雷区预警
- 哪些回复方式**绝对不能用**
- 对方可能的应对方式预判

## 输出格式
```
📊 情绪仪表盘
├── 显性: [情绪标签] (强度 X/10)
├── 隐性: [推断的潜台词]
└── 趋势: [升温/降温/稳定]

🎯 意图分析
└── [对方的真实意图，1-2 句话]

💬 回复方案
├── 🟢 安全选项: [回复内容]
├── 🔵 进攻选项: [回复内容]
└── 🟣 战略选项: [回复内容]

⚠️ 雷区
└── [绝对不要xxx]
```
